# Murojaat Bot

## Kirish
Murojaat bot - bu foydalanuvchilarga murajaat yo'llash mumkin bo'lgan interaktiv bot.

## Talablar
1. Python 3.8 yoki undan yuqori versiya.
2. `python-telegram-bot` kutubxonasi.

## O'rnatish
Kodni `git clone` yordamida yuklab oling:
```bash
git clone https://github.com/Salohiddin1675/murojaat_bot.git
```

## Foydalanish
1. Bot tokenini oling va `config.py` faylida joylashtiring.
2. Botni ishga tushirish uchun quyidagi buyruqni bajarish:
```bash
python main.py
```

## Funksiyalar
- Foydalanuvchilar bilan interaktiv muloqot.
- Murojaatlar va xabarnomalar.

## Hujjatlar
Qo'shimcha ma'lumot uchun loyiha hujjatlarini o'qing.
